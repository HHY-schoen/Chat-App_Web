## Chat-App_Web ( live chat room )
建立一個線上即時聊天室，用戶可以再裡面即時互動、傳遞訊息。

使用 Flask, flasksio 和其他 module 創建一個 front-end，透過 socket server 溝通和傳遞訊息。

## Old Message Server
使用 SocketIO 前，使用 Python sockets 編寫了一個自定義的消息伺服器 ( 程式碼位於 old_msg_server/ )
